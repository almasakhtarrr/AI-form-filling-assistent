# AI-form-filling-assistent
Indian Citizen Services Assistant 🇮🇳
Built by a 1st-year Engineering student with the assistance of Google Gemini AI. This application automates the processing of Indian ID cards using OCR and intelligent data extraction.

🚀 Project Overview
This project serves as a bridge between raw document images and structured data. It handles the complexities of real-world OCR noise and provides a clean user interface for document management.

Key Features:
Automated OCR: Extracting text from Indian ID cards using Tesseract and OpenCV.
Smart Extraction: Utilizing Lookaround Regex to accurately capture Virtual IDs (VID) and other sensitive fields.
Professional UI: A web-based dashboard for file uploads and data downloads built with Streamlit.
